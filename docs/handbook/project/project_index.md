# The Raptoreum Project

Raptoreum (RTM) is a decentralized, Proof-of-Work cryptocurrency designed to bring smart contracts, asset creation, and enhanced security to a flexible blockchain environment.

The name Raptoreum is derived from the Victorian term for a bird of prey and ium/eum “place for” or “pertaining to” in this case birds of prey.

## Release and Origins

- Start 2019 as Fork of Raven Coin
- Mainnet Launch: February 26, 2021
- Codebase: Fork of Dash, which itself derives from the Bitcoin (BTC) code.
- Consensus Algorithm: GhostRider — a custom algorithm combining several mining hash functions to improve ASIC resistance and accessibility for CPU miners.


## Core Specifications

- Ticker Symbol: RTM
- Decimals: 8
- Maximum Supply: 21,000,000,000 RTM
- Block Time: ~2 minutes
- Block Reward: <5,000 RTM per block (including a regular reduction)
- Reward Distribution: 75% miners, 20% SmartNodes, 5% Developper
- Smartnodes Collateral: 1.8 million RTM
- Difficulty Adjustment: Every block


## Key Features

- Futures Transactions
- Assets: Token and asset creation directly on the blockchain.
- Private IFPS Nertwork for Assets
- Enhanced Security: Built-in protection against 51% and double-spend attacks.
- Smartnodes Network: Rewards for operators who provide stability and speed to the network.
- Future Support for Smart Contracts: Raptoreum aims to integrate smart contract execution via a dedicated virtual machine.


## Ecosystem and Compatibility

Raptoreum is compatible with common cryptocurrency infrastructure, including:

- Standard wallets for Windows, Linux, macOS, and ARM 
- Integration with major mining software and mining pools
- Explorer tools, community governance, and DAO development roadmaps
