## help

List all commands, or get help for a specified command.

### Arguments

| Position | Name | Type | Required | Default | Description |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | `command` | string | Optional | all commands | See CLI help for details |
| 2 | `subcommand` | string | Optional | all subcommands | See CLI help for details |


### Result
```json
"str"    (string) The help text
```
