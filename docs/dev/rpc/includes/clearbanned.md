## clearbanned

Clear all banned IPs.

### Arguments

No arguments.

### Result
```json
null    (json null)
```

### Examples
```bash
 raptoreum-cli clearbanned
```
```bash
 curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "clearbanned", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:10225/
```
